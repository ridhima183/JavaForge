A little Java garden that grows from the very first `public static void main` all the way up to real, database-talking applications. 🌿

Everything you'd want to learn Java — properly — is tucked inside:

---

### 💭 Why this exists

> Most tutorials teach syntax. Few teach *why* Java is shaped the way it is. This repo is my little attempt to close that gap — one runnable example at a time. 🌸

- 📖 Concept-first, code-second
- 🧪 Practice-driven, not just theory
- 🎯 Built to double as interview revision material
